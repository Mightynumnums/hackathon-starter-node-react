## Hackathon Starter: Node.js + React

To run in dev mode, run both projects separately:

Frontend:
```
# from root
cd frontend
yarn start
```

http://localhost:3000

Backend:
```
# from root
yarn start-dev
```

API:
http://localhost:3001/api

#

In prod mode:

```
# from root
cd frontend
yarn build
cd ..
yarn start
```

http://localhost:3001
